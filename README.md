### 简易说明
按照网上的教程自己做了一个演示软件。
功能是点击新建可以在下面创建新的记录，可以勾选。
可以鼠标划过某一个记录后面会出来删除按钮。
可以点击删除。
也可以是勾选多个，然后批量删除。
勾选的数量和总数在最后一行有显示。
